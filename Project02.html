Employee Management System

The Employee Management System is a web application developed in C# using Visual Studio Code. It enables organizations to efficiently manage employee records, track attendance, process payroll, and evaluate performance.

Technologies Used: C#, ASP.NET Core, SQL Server, Entity Framework, Bootstrap

Key Features:

  • Employee Registration and Profile Management
  • Attendance Tracking and Shift Management
  • Payroll Calculation and Salary Processing
  • Role-Based Access Control for Admins and Employees
  • Performance Reviews and Reports
